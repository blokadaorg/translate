Blokada，广告屏蔽器

如果您想 有效屏蔽广告、跟踪器和恶意警告，如果想要保存您的数据计划，请更快地使用设备，并且保护您的隐私，Blokada 是您使用的。 它是 免费，安全和开源。

Blokada在浏览器 所有应用，而不仅仅是浏览器！ 它无需root，支持 wifi 和移动网络，并且完全安全，简单。 只需要打开它，它将会静悄地做工作！

我们的故事

Blokada开发者Karol Gusak于2016年11月首次发布。 目标是保持设备无广告，避免使用不必要的移动数据的不需要内容。 第一个功能和错误修复很快被发运，著名的 XDA 论坛上的应用程序主题每小时更新。 为了使用户更容易获得和协助，项目在Telegram上打开了门户。 前两名成员是卡尔莫和论坛成员之一，即Péter Róth。 后来，他开始翻修广告屏蔽器的Facebook网页和网站，同时也帮助快速发展的聊天室。

为了建立一个帮助翻译应用、创建内容、测试和调试测试版以及管理社会页面的组，Blokada Insiders于2018年创建，它积极努力使Blokada成为一个好事。 您知道今天Blokada是否可以在25种语文中获得？
Sebastian Januszevski 加入 Telegram 频道的管理，协助成员并积极贡献应用程序测试，复制用户报告。 根据他的反馈和知识，Blokada获得了更加稳定和可靠的。

卡罗尔出生在波兰。 在瑞典花费了几年时间之后，他现在回到华沙，作为开发者参与各种流动项目。 你并不感到惊讶，你吗？
Péter 来自匈牙利，位于匈牙利首都布达佩斯，他在那里担任信息技术部门工程师，了解大数据技术，协助全世界已知公司实现其目标。
塞巴斯蒂安，在布宜诺斯艾利斯工作，他在那里教导大学的统计和数据处理。 他是PHP 和 JavaScript 中的一个令人惊心的开发者，一些 WordPress 插件被发布在他的名称下。

自开始以来，Blokada已经从手机到桌面和智能电视上下载和安装在200 000多台设备上。

我们的远景是什么？ 团队将努力支持更广泛的设备，使布隆迪正在运行时连接到真正的 VPN 服务，并保护您的隐私，为您提供更多关于您受到保护的广告的信息。
