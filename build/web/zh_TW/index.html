Android 上的最佳廣告阻擋程式：免費且開放原始碼。

Blokada，廣告阻擋程式

如果您想要有效地阻擋廣告、追蹤器、惡意軟體、節省您的行動數據消耗、為您的裝置加速並保護您的隱私，而且只使用一個應用程式就達到以上目的，那麼 Blokada 就是為您所設計的。 它免費、安全且開源。

Blokada 在您所有的應用程式上都能運作，不只有瀏覽器！ 其不需要 root 權限，同時支援Wi-Fi 與行動網路，其也完全安全且易於使用。 只要把它打開，它就會默默地把工作做好！

我們的故事

Blokada 最初於 2016 年 11 月釋出，開發者為 Karol Gusak。 目標是讓裝置不受廣告污染，避免在不想要的內容上耗費非必要的行動數據。 第一個功能與錯誤修復很快地推出，在知名的 XDA 論壇上的應用程式討論每個小時都有更新。 為了讓使用者更方便，並可以更容易地協助他們，專案在 Telegram 上開了群組。 前兩位成員是 Karol 與論壇的其中一個成員，Péter Róth。 後來他除了協助管理快速成長的聊天室，也開始翻修了廣告阻擋程式的 Facebook 頁面與網頁。

為了建構可以協助翻譯應用程式、建立內容、測試與為測試版本除錯、管理社群頁面，Blokada Insiders 在2018年建立，它努力讓 Blokada 愈來愈好。 您知道嗎？現在 Blokada 已經翻譯成 25 種語言了。
Sebastian Januszevski 加入 Telegram 頻道管理，協助成員並在測試應用程式、重現使用者回報方面有相當活躍的貢獻。 根據他對問題的反饋與本身的技能，Blokada 變得更穩定而可靠。

Karol 在波蘭出生。 在瑞典待了數年，他回到了華沙，他在那裡以開發者的身份參與許多行動裝置的專案。 您並不感到驚訝，對吧？
Péter 來自匈牙利，住在其首都布達佩斯隔壁，他在那裡是資訊產業的工程師，了解大數據的深層領域，協助全球知名的公司達成他們的目標。
Sebastian 在布宜諾斯艾利斯生活與工作，他在那裡的大學教授統計與資料處理。 他是 PHP 與 JavaScript 的興趣使然的開發者，並以他的名字發佈了一些 WordPress 外掛程式。

自從那時候，Blokada 在超過二十萬臺裝置上被下載。不管從手機到平板電腦，甚至在智慧型電視上都有他的蹤跡。

我們還有什麼目標呢？ 團隊將會支援更廣泛的裝置、讓您可以在 Blokada 執行時連線到真的 VPN ，以更加保護您的隱私，為您提供更多阻擋了哪些廣告的資訊。
