Welcome and thank you for using Blokada DNS!

Blokada is free and open source and it respects your privacy. By using it you agree to the Terms of Service and Privacy Policy, which you can always find here.

We would like to assist you with your first steps:

  1. Default DNS servers are used currently. You can change it by tapping on Default DNS icon.
  2. Blokada is set to start when your phone is started up. You can turn if off by tapping on Start on boot.
  3. Blokada will be running only when you are connected to the internet. You can change it by setting Online only to off.
  4. In the Whitelist you can define which apps shouldn't use your DNS servers. They will use default device DNS servers instead.
  5. Tapping in the top right corner you can find more options and the Help menu where you can get detailed information about the different options in Blokada.
