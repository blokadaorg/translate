Willkommen und danke für die Verwendung von Blokada DNS!

Blokada ist kostenlos, Open Source und respektiert Ihre Privatsphäre. Durch die Nutzung stimmen Sie den Nutzungsbedingungen und der Datenschutzrichtlinie zu, die Sie immer hier finden können .

Wir möchten Ihnen bei Ihren ersten Schritten helfen:

  1. Standard-DNS-Server werden derzeit verwendet. Sie können dies ändern, indem Sie auf das Symbol Standard-DNS tippen.
  2. Blokada wird gestartet, wenn das Telefon gestartet wird. Sie können dies ausschalten, indem Sie auf Start beim Booten tippen.
  3. Blokada wird nur ausgeführt, wenn Sie mit dem Internet verbunden sind. Sie können dies ändern, indem Sie Nur Online auf Aus setzen.
  4. In der Whitelist können Sie festlegen, welche Apps Ihre DNS-Server nicht verwenden sollen. Die App nutzt stattdessen die Standard DNS Server des Geräts.
  5. Wenn Sie in der oberen rechten Ecke tippen, finden Sie weitere Optionen und das Menü Hilfe , in dem Sie detaillierte Informationen zu den verschiedenen Optionen in Blokada erhalten.
