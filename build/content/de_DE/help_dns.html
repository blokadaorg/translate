Eine App funktioniert nicht wie erwartet.

Wenn Sie sehen, dass eine App nicht wie erwartet funktioniert, den Inhalt nicht lädt oder ein ähnliches Fehlverhalten auftritt, gehen Sie bitte zum Hauptbildschirm, wählen Sie am unteren Bildschirmrand Whitelist Tippen Sie auf das + Zeichen, wählen Sie App und geben Sie seinen Namen ein. Tippen Sie am Ende auf Speichern. Damit filtert Blokada die URLs in dieser App nicht.

Ich kann Blokada nicht starten, nachdem ich es installiert habe.

Für diese Situation gibt es zwei mögliche Ursachen. Überprüfen Sie zuerst, ob Sie eine aktive VPN-Verbindung haben und ob die Option Always-on VPN aktiviert ist. Um dies zu tun, gehen Sie zu den Einstellungen Ihres Telefons, suchen Sie im Bereich Netzwerk nach VPN, tippen Sie auf das Zahnrad neben dem vorhandenen VPN. Deaktivieren Sie es, trennen Sie die Verbindung zur aktiven VPN-Verbindung und versuchen Sie Blokada neu zu starten.

Wenn Sie keine andere VPN-Verbindung haben, checken Sie in Blokada ein, wenn Nur Online aktiviert ist, und wenn ja, überprüfen Sie, ob Sie mit dem Internet entweder über WLAN oder mit mobilen Daten verbunden sind. Blokada startet nicht, wenn nur Online aktiviert ist und Sie keine Internetverbindung haben.

Wenn es nicht hilft, starten SIe Ihr Handy neu, dies hilft manchmal.

Blokada wird nich einiger Zeit deaktiviert.

Aktivieren Sie als ersten Schritt Keep alive in Blokada. In diesem Fall wird Blokada versuchen, online zu bleiben, aber bitte beachten Sie, dass Ihr System dieses Verhalten weiterhin beeinflussen kann. Wenn Sie die einstellung vorgenommen haben und immer noch sehen, dass Blokada immer wieder angehalten wird, überprüfen Sie die Systemeinstellungen und schalten Sie jede Batterie- und Speicheroptimierung für Blokada aus. Leider ist es in jedem System anders, daher können wir nicht sagen, wo wir es machen sollen. Bitte überprüfe deine Einstellungen. Eine andere Quelle für ein solches Verhalten kann sein, dass Sie einen memory cleaner verwenden. Bitte setzen Sie Blokada auf die Whitelist oder Ausschlussliste. Wenn diese Schritte in Ihrem Fall nicht hilfreich sind, melden Sie bitte einen Fehler (siehe unten).

Wie funktioniert Blokada?

Blokada ist eine dünne Schicht zwischen der Netzwerkschnittstelle (Wlan oder Handy) Ihres Telefons und den Anwendungen, die Sie verwenden. Um DNS-Server-Adressen Ihres Geräts ändern zu können, wird eine VPN-Verbindung hergestellt. Blokada sendet Ihre Daten nicht an einen anderen Ort. Das VPN wird nur zum Erstellen einer lokalen Verbindung verwendet, damit Apps denken, dass sie unterschiedliche DNS-Server verwenden sollten. Blokada DNS filtert Ihre Verbindung in keiner Weise, aber abhängig von den DNS-Servern, die Sie auswählen, können sie dies tun.

Kann ich ein anderes VPN verwenden, während Blokada eingeschaltet ist?

Leider nein, es ist eine Systembeschränkung; Sie können jeweils nur eine aktive VPN-Verbindung haben. Wenn Sie sich einem anderen Dienst anschließen, wird Blokada beendet.

Welche Optionen gibt es in Blokada?

Wie Sie sicher gesehen haben, hat Blokada einige Optionen auf dem Hauptbildschirm, die beim ersten Start nicht erklärt werden. Dies holen wir jetzt nach. Sie können Folgendes sehen:

  • Whitelist: the whitelist is the exception list, here you can define apps where Blokada won’t change DNS for. Blokada wird standardmäßig mit mehreren Whitelist-Apps installiert.
  • Keep alive: Mit dieser Option wird sichergestellt, dass Blokada nicht pausiert oder vom System angehalten wird. Bitte beachten Sie, dass Ihr System es überschreiben kann, stellen Sie sicher, dass Ihre Einstellungen Blokada im Hintergrund laufen lassen.
  • Nur online: Wenn diese Option aktiviert ist, wird Blokada beendet, wenn keine aktive Verbindung besteht, und erneut starten, sobald die Verbindung wieder hergestellt wird. Wenn deaktiviert, läuft Blokada immer im Hintergrund (auch wenn kein Netzwerk verfügbar ist). Bitte beachten Sie, dass diese Option möglicherweise nicht auf jedem Gerät funktioniert.
  • Start beim Booten: Wenn diese Option aktiviert ist, wird Blokada nach dem Hochfahren des Systems gestartet. Bitte beachten Sie, dass es für diese Option Konfigurationen auf Systemebene gibt. Daher kann nicht garantiert werden, dass Blokada beim Booten gestartet werden kann. In diesem Fall überprüfen Sie bitte die Einstellungen Ihres Telefons. Die Standardkonfiguration ist aktiviert. Wenn Sie Blokada manuell starten möchten, deaktivieren Sie diese Option.

Nicht konfigurierbare Dinge:

  • Aktuell: Wenn Sie dies sehen, benutzen Sie die neueste Version von Blokada. Halten Sie die App auf dem neuesten Stand, da sie weniger Bugs und mehr Funktionen enthält!

Ich kann kein Update installieren. Was kann ich tun?

Überprüfe zuerst, ob die Android-Version die Sie verwenden Version 5 oder neuer ist. Android 4 und ältere Versionen werden in Blokada 3.0 und neueren Versionen nicht unterstützt. Um dies zu tun, gehen Sie zu den Einstellungen Ihres Telefons, gehen Sie zu Über Telefon und suchen Sie nach Android-Version

Wenn auf Ihrem Telefon Andriod 5 oder neuer ausgeführt wird, starten Sie das Gerät neu. Es klingt seltsam, aber manchmal hilft es. Wenn das Problem dadurch nicht behoben wurde, deinstallieren Sie Blokada und installieren Sie es erneut. Es kann sein, dass Ihr Gerät nicht unterstützt wird. Bitte überprüfen Sie die Gerätedatenbank.

Wenn keiner dieser Schritte geholfen hat, treten Sie bitte unserem Telgramm-Chat auf go.blokada.org/chat bei, um Hilfe zu erhalten, oder melden Sie einen Fehler (siehe unten).

Kann ich die VPN-Benachrichtigung ausblenden?

Unglücklicherweise nicht. Einige Geräte zeigen möglicherweise eine störende VPN-Benachrichtigung an, während VPN aktiv ist, und es ist nicht möglich, sie zu verbergen. Versuchen Sie, Ihr Gerät zu aktualisieren, da neuere Android-Versionen dieses Problem nicht aufweisen.

Wie melde ich ein Problem, eine Idee oder einen Vorschlag?

Wenn Sie Probleme haben, besuchen Sie bitte unseren Telegram-Chat unter go.blokada.org/chat, wo unsere Community Ihnen gerne weiterhilft. Alternativ wählen Sie einfach die Feedback-Option aus dem Menü in der oberen rechten Ecke und folgen Sie den Anweisungen. Bitte geben Sie so viele Informationen wie möglich und die Schritte die Sie unternommen haben, um uns zu ermöglichen das Problem besser reproduzieren zu können - es hilft sehr bei der Lösung Ihres Problems! Wenn Sie ein Pro-User sind, können Sie direkt zum Issue Tracker von GitHub gehen.

Kann Blokada meine privaten Daten sehen?

Nein. Blokada liest oder manipuliert Ihre privaten Informationen nicht. Blokada DNS überwacht Ihre Daten in keiner Weise, sondern lässt Ihre App einfach denken, dass sie andere DNS-Server verwenden sollten. Blokada hat keinen Zugriff auf Ihren verschlüsselten Datenverkehr.

Verwendet Blokada viel Batterie?

Nein. Je nach den vorgenommen Einstellungen ist der Akkuverbrauch minimal, und bei den meisten neuen Geräten, besonders bei Marshmallow (Android 6) oder neuer, werden Sie kaum einen Batterieverbrauch bemerken. Bei einigen unglücklichen Geräten kann es jedoch zu erhöhtem Batterieverbrauch kommen. Einzelheiten finden Sie in der Gerätedatenbank.

Wird Blokada mein Gerät verlangsamen?

Nein. Abhängig von Ihren Einstellungen kann Blokada ziemlich effizient sein. In der Tat können sowohl die Leistung als auch die Akkulaufzeit Ihres Geräts sichtbar verbessert werden, da Blokada Ihr Gerät davon abhält, Inhalte herunterzuladen und zu verarbeiten, die Sie nicht benötigen oder sich nicht darum kümmern. Benutzer mit älteren Geräten können jedoch einen Leistungsabfall erleiden. Einzelheiten finden Sie in der Gerätedatenbank.

Ist Blokada wirklich frei zu benutzen?

Ja! Blokada begann als mein Hobby-Projekt, ist noch größtenteils von mir entwickelt. Es ist meine Art, der Gemeinschaft etwas zurückzugeben. Blokada ist vollständig Open Source und wird für immer kostenlos bleiben. Das heißt, dass die Entwicklung sehr viel Zeit in Anspruch nimmt. Bitte denken Sie daran, das Projekt auf jede erdenkliche Weise zu unterstützen (siehe unten).

Blokada ist großartig, wie kann ich dem Projekt helfen?

Es gibt viele Möglichkeiten, Blokada finanziell oder anderweitig zu unterstützen. Alle Hilfsmittel sind sehr wichtig für das Projekt. Bitte gehen Sie auf diese Seite, um weitere Informationen zu erhalten oder nur um an unserem Chat teilzunehmen.

Ich habe eine andere Frage, wo kann ich das fragen?

Wenn Ihre Frage hier nicht beantwortet wird, treten Sie bitte unserem Chat-Kanal auf dem go.blokada.org/chat Link bei, unsere Community wird Ihnen helfen. Wenn es Ihnen nicht passt, verwenden Sie die Option Feedback im oberen Menü. Wenn Sie eine eher technische Frage haben, gehen Sie bitte direkt zum Tracker von GitHub.
