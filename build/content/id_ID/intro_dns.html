Selamat datang dan terima kasih untuk menggunakan Blokada DNS!

Blokada gratis dan open source dan menghormati privasi Anda. Dengan menggunakannya, Anda menyetujui Ketentuan Layanan dan Kebijakan Privasi, yang selalu dapat Anda temukan di sini.

Kami ingin membantu Anda dengan langkah pertama Anda:

  1. DNS server standar yang digunakan saat ini. Anda dapat mengubahnya dengan menekan ikon Standar DNS.
  2. Blokada diatur untuk memulai saat telepon Anda dinyalakan. Anda dapat mematikan jika dengan mengetuk Mulai saat boot.
  3. Blokada akan berjalan hanya ketika Anda terhubung ke internet. Anda dapat mengubahnya dengan menyetel Hanya online ke posisi mati.
  4. Di Whitelist Anda dapat menentukan aplikasi mana saja yang tidak diperbolehkan memakai server DNS Anda. Mereka akan menggunakan server DNS standar perangkat.
  5. Mengetuk di sudut kanan atas Anda dapat menemukan lebih banyak opsi dan menu Bantuan tempat Anda dapat memperoleh informasi mendetail tentang berbagai opsi di Blokada.
